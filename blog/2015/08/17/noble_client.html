Nodule Command Line App

on 17 Aug 2015

The nodule.op command line app is now available - you can get the full setup details from the nodule_noble github repository.

This means that you no longer need the nodule.io mobile app if you want to route your Bluetooth Low Energy devices out to the nodule server.

The command line app should be compatible with any platform that can run node and the noble npm package.

This includes - OSX, Windows, Linux, RaspberryPI, Edison.

For the RaspberryPI all you'll need is a Bluetooth 4.0 dongle.

The command line app works very well on OSX. It also works well on the RaspberryPI (though you may need to do a bit of work if your Bluetooth dongle is not supported out of the box - see here for some suggestions).

Installation on Windows requires a bit more - see the README file on github for more information.
